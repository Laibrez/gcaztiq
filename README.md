# Caztiq — Global Creator Payouts

Caztiq is a platform for brands to manage and pay creators.

**URL**: [caztiq.com](https://caztiq.com)

The payout infrastructure layer for the creator economy. Brands pay creators based on performance, not guesswork.

## Features

- **Wallet System:** Top up your balance via Stripe.
- **Bulk Payouts:** Pay hundreds of creators at once via CSV upload.
- **Creator Management:** Invite and track your creator roster.
- **Tax Compliance:** Collect tax forms from creators before they get paid.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Express, Node.js, Stripe
- **Database:** Supabase (PostgreSQL + Auth)
- **Deployment:** Vercel (Frontend), Railway (Backend)

## Local Development

1. Clone this repository.
2. Run `npm install`.
3. Create a `.env` file based on the environment variables needed (VITE_API_URL, etc.).
4. Run `npm run dev`.
